Aim Of This Project:

Making Rest API for NLTK to use NLTK methods, Creating web based application for online NLTK.

Summary:

First of all nltk data downloaded and nltk installed on local computer.After nltk methods which will be in our website were chosen and for some of them turkish language libraries were added.Then website coding was finished and website deployed.Now with website and Api everybody can use it easily without downloading anything.

Usage:

Choose an option from left side menu and enter text or upload file that you want to see the result of NLTK and execute.
